# stack-game
My shippable game, hij is nog niet helemaal shippable, der zit wel al core gameplay in en een score counter, wat ik later nog wil toevoegen is een achtergrond gericht naar mijn doelgroep,
mijn doelgroep is dan ook een beetje kinderen rond 10/11 jaar, aangezien zij meestal maar korte tijd op hun mobiel kunnen zitten als ze deze hebben, vandaar deze doelgroep, eventuele verbeteringen die ik 
nog zou willen toevoegen zijn, dat je game kan hertstarten als je "verliest", een highscore counter, en misschien nog een hoofd menu. 


## Build
Open project with Unity Editor, choose Scene 'Game'.
